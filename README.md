# Little Automations

A compilation of automation python scripts made for practice.
</br>

It includes:
- Renaming: Renaming all files in folder.
- Organizer: Moving every file in a folder to it's apropriate folder inside the special folders.
- More will be added.

Inside the folders there is a executable file and the source code written in python.
